1) What is the difference between var, let, and const?
var is the old way to declare variables, can change.

let is the newer to declare variables, can change.

const is for fixed values.

2) What is the difference between map(), forEach(), and filter()? 
forEach() runs a function on each item, doesn’t return anything.

map() runs a function on each item and returns a new array with the results.

filter() checks each item and returns a new array with only the items that pass the test.

3) What are arrow functions in ES6?
Arrow functions in ES6 are a short way to write functions using ()=> instead of function

4) How does destructuring assignment work in ES6?
Destructuring in ES6 is a simple way to take values from an array or object and put them into variables.

5) Explain template literals in ES6. How are they different from string concatenation?
Template literals in ES6 let strings easily and add variables inside using ` and ${}

string concatenation: "total " + price 
template literals: `total ${price}`
